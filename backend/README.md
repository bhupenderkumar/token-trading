# Backend API for Solana Token Management

This directory contains the Node.js backend for the Solana Token Management Dashboard. It provides API endpoints to support the frontend application.

## Features

- **API Server**: An Express.js server that handles requests from the frontend.
- **Solana Interaction**: Communicates with the Solana blockchain for token operations.
- **Swagger Documentation**: Includes Swagger for API documentation and testing.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run the server**:
    ```bash
    npm start
    ```

The server will start on `http://localhost:3001`.
